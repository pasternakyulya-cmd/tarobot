1. Делаешь изменения, отмечаешь галочкой в "Commit" слева файлы которые надо закинуть (обычно все)
2. Нажимаешь снизу Commit and Push, пишешь Commit Message, потому что он обязательный
3. Убеждаешься, что разлилось на гитхаб
4. Нажимаешь Win+R, вводишь cmd
5. Вводишь "ssh root@81.90.31.243"
6. Попросит пароль, вставляешь его со страницы с ВПС в консоль Ctrl+V
7. Когда напишет root@цифры, значит все хорошо
8. Вводишь следующие команды по очереди


---

cd tarobot
git pull
docker stop my-app-container
docker rm my-app-container
docker build -t my-app .
docker run -d --name my-app-container my-app

9. Чтобы проверить, что ошибок нет, вводишь

docker logs my-app-container

Если после enter ничего не выводит - все работает!
